# Chat with CSV

## Project Overview

This project provides a web-based application that allows users to upload CSV files and interact with their data through natural language queries. It leverages a Retrieval Augmented Generation (RAG) pipeline to provide intelligent responses based on the content of the uploaded CSVs. The application features a FastAPI backend, a LangChain-powered RAG system, and a simple, intuitive frontend.

## Features

*   **CSV Upload:** Easily upload your CSV files through a user-friendly web interface.
*   **Natural Language Querying:** Ask questions about your CSV data in plain English.
*   **Intelligent Responses:** Get accurate and context-aware answers powered by a RAG pipeline.
*   **Scalable Backend:** Built with FastAPI for high performance and easy extensibility.
*   **Robust RAG Pipeline:** Utilizes LangChain, Groq, HuggingFace Embeddings, and FAISS for efficient data retrieval and generation.
*   **Simple Frontend:** A clean HTML, CSS, and JavaScript interface for seamless interaction.

## Technologies Used

**Backend:**

*   **FastAPI:** A modern, fast (high-performance) web framework for building APIs with Python 3.7+ based on standard Python type hints.
*   **LangChain:** A framework for developing applications powered by language models.
*   **Groq:** Used for fast inference with large language models (LLMs).
*   **HuggingFace Embeddings:** Provides state-of-the-art text embeddings for semantic search.
*   **FAISS:** A library for efficient similarity search and clustering of dense vectors.
*   **Python-dotenv:** For managing environment variables.

**Frontend:**

*   **HTML5:** Structure of the web pages.
*   **CSS3:** Styling of the web pages.
*   **JavaScript:** Interactivity and dynamic content.

## Setup and Installation

To get this project up and running on your local machine, follow these steps:

### Prerequisites

*   Python 3.8+
*   `pip` (Python package installer)

### 1. Clone the Repository

```bash
git clone https://github.com/zakir-maswani/chat_with_csv.git
cd chat_with_csv
```

### 2. Create a Virtual Environment (Recommended)

```bash
python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure Environment Variables

Create a `.env` file in the root directory of the project and add your Groq API key:

```
GROQ_API_KEY="your_groq_api_key_here"
```

You can obtain a Groq API key from [Groq Console](https://console.groq.com/).

### 5. Run the Application

```bash
uvicorn main:app --reload
```

The application will be accessible at `http://127.0.0.1:8000`.

## Usage

1.  Open your web browser and navigate to `http://127.0.0.1:8000`.
2.  Use the provided interface to upload your CSV file.
3.  Once the CSV is uploaded, you can type your questions into the chat interface.
4.  The RAG pipeline will process your query and the CSV data to provide an answer.
